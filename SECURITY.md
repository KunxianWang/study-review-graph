# Security Policy

## Supported Versions

This project is early-stage. Security fixes will be applied to the latest development line first.

## Reporting a Vulnerability

Please do not open public issues for security-sensitive reports.

Instead, contact the maintainers privately with:

- a description of the issue
- affected files or modules
- reproduction steps if available
- potential impact

We will acknowledge receipt as soon as practical and work with you on coordinated disclosure.

## Scope Notes

This repository processes user-provided study materials. Please report issues related to:

- unsafe file handling
- prompt or retrieval leakage across materials
- path traversal risks
- insecure dependency or credential handling
