# Review Material Skill

This skill packages a repeatable workflow for turning lecture materials into high-quality study notes.

Included:
- `SKILL.md`: the skill manifest and operating instructions
- `assets/output-templates.md`: reusable response shapes
- `references/design-notes.md`: rationale for the workflow

Suggested use cases:
- “根据这份 PPT 生成复习资料”
- “详细讲一下这里面的算法，再给我一个手算例子”
- “按考试复习的方式整理这章内容”
- “先讲概念，再做例题演算”
